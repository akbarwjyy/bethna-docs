# Glossary

## Options Terms

**ATM (At-The-Money)**: Option strike price equals current asset price.

**Call Option**: Right to buy an asset at a specific price.

**Delta**: Rate of change in option price relative to underlying asset price.

**Expiry**: Date when option contract expires.

**Gamma**: Rate of change in delta.

**Implied Volatility (IV)**: Market's expectation of future price volatility.

**ITM (In-The-Money)**: Option with intrinsic value (call: strike < price, put: strike > price).

**OTM (Out-of-The-Money)**: Option with no intrinsic value.

**Premium**: Price paid to buy an option.

**Put Option**: Right to sell an asset at a specific price.

**Strike Price**: Price at which option can be exercised.

**Vega**: Sensitivity to implied volatility changes.

## DeFi Terms

**AMM (Automated Market Maker)**: Decentralized exchange using liquidity pools.

**DEX**: Decentralized exchange.

**Liquidity Pool**: Pool of tokens used for trading on AMMs.

**Non-Custodial**: Users maintain full control of their funds.

**Slippage**: Difference between expected and actual execution price.

**TVL (Total Value Locked)**: Total capital deposited in a protocol.

**Vault**: Smart contract that holds user funds for automated strategies.

## Blockchain Terms

**Base Network**: Ethereum L2 using OP Stack.

**EIP-4337**: Account Abstraction standard.

**ERC-20**: Fungible token standard on Ethereum.

**Gas**: Transaction fee paid to execute on blockchain.

**L2 (Layer 2)**: Scaling solution built on top of Ethereum.

**Multisig**: Wallet requiring multiple signatures to execute transactions.

**Smart Contract**: Self-executing code on blockchain.

**Wallet**: Software or hardware storing private keys for blockchain accounts.
