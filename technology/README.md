# Technology Stack

## Overview

BethNa uses modern, battle-tested technologies across frontend, backend, blockchain, and AI layers.

## Tech Stack Summary

| Layer | Technologies |
|-------|-------------|
| **Frontend** | Next.js 15, React 18, TypeScript, Tailwind CSS |
| **Backend** | Python 3.11, FastAPI, PostgreSQL, Redis |
| **Blockchain** | Solidity 0.8.20, Foundry, Base Network |
| **AI/ML** | LangChain, scikit-learn, TensorFlow |
| **Infrastructure** | Vercel, Railway, Supabase |

---

## Frontend Stack

### Core Framework
- **Next.js 15 (Turbo)**: React framework with SSR
- **React 18**: UI component library
- **TypeScript 5.3**: Type-safe development

### Styling & Design
- **Tailwind CSS 3.4**: Utility-first styling
- **Framer Motion 11.0**: Animations
- **Radix UI**: Headless components

### Web3 Integration
- **Wagmi 2.0**: React hooks for Ethereum
- **Viem 2.0**: TypeScript Ethereum library
- **WalletConnect**: Multi-wallet support

### State Management
- **Zustand**: Lightweight state management
- **React Query**: Server state management
- **Jotai**: Atomic state management
